# Steam Store Card Prices

Steam 商店详情页用的油猴脚本。打开类似 `https://store.steampowered.com/app/805940/` 的页面时，脚本会显示该游戏集换式卡牌的出售价格、卖家到手价和求购价格。

当前脚本版本：`0.6.9`

## 功能

- 从当前商店页 URL 读取游戏 `appid`。
- 查询 Steam 社区市场 `appid=753` 下的当前游戏卡牌。
- 支持普通卡和闪卡分别查询、分别显示。
- 支持出售价、到手价、求购价分别显示。
- 支持右侧悬浮、左侧悬浮、页面内三种面板位置。
- 支持悬浮面板宽度和高度自定义。
- 结果按查询配置缓存 5 分钟，点“刷新”可立即重新读取。

## 查询方式

基础卡牌列表使用 Steam 市场搜索接口：

- `category_753_Game[]=tag_app_{appid}`
- `category_753_item_class[]=tag_item_class_2`
- `category_753_cardborder[]=tag_cardborder_0` 查询普通卡
- `category_753_cardborder[]=tag_cardborder_1` 查询闪卡

如果开启求购价查询，脚本会逐张读取 Steam 市场订单簿接口，解析最高求购价和求购数量；遇到临时网络错误会自动重试，并在接口不可用时回退解析市场详情页。(若开启求购价查询可能会比较慢，非必要不建议开启）

## 到手价

到手价按 Steam 社区市场常见手续费规则计算：

- 优先读取 Steam 钱包参数 `g_rgWalletInfo`。
- 按 Steam 手续费、发行商手续费、最低手续费计算卖家收入。
- 对人民币、港币等地区会套用单独的地区规则：Steam 手续费和发行商手续费分别最低按 `0.07` 计算；脚本会结合钱包地区信息和价格文本识别这类规则。
- 支持部分货币的 `round` / `floor` 手续费规则。
- 如果开启求购价查询，会从市场详情页读取实际 `publisherFeePct` 校正发行商手续费。
- 对印度卢比这类 Steam 不显示小数的币种，会根据 `sell_price_text` 推断内部价格单位和最低手续费。

## 设置

脚本面板内有“设置”区域。

- 查询普通卡、查询闪卡：控制是否访问对应卡牌列表。
- 查询普通卡求购价、查询闪卡求购价：控制是否补充最高求购价。
- 显示普通卡、显示闪卡：只控制界面展示，不影响后台是否查询。
- 显示出售价、显示到手价、显示求购价：只控制价格列展示。
- 面板位置：右侧悬浮、左侧悬浮、页面内。
- 面板宽度、面板高度：悬浮模式下按像素调整。

查询开关变化会刷新数据；显示和面板设置变化只重画当前结果。

## 安装

1. 打开 Tampermonkey。
2. 新建脚本。
3. 粘贴 `steam-store-card-prices.user.js` 的内容并保存。
4. 打开 Steam 商店游戏详情页。
